# Usage
macOS or Linux
```console
$ ./AllFolders.sh 3
```
DOS
```console
C:\> AllFolders.bat 3
```
PowerShell
```console
PS C:\> .\AllFolders 3
```
Where '3' can be up to any 3 digits.
The scripts are zero based. Based on the above commands, the following folder structure will be created.

```
├── EDIT
│   ├── EXPORTS
│   ├── Footage
│   ├── Resolve
│   └── Thumbnail

│   │   ├── Geo

│   │   ├── Scenes
│   │   ├── Sims
│   │   └── Textures
│   └── Nuke
│       ├── Comps
│       ├── Live_Action
│       ├── Precomps
│       └── Scripts
├── sh001
│   ├── Blender
```
